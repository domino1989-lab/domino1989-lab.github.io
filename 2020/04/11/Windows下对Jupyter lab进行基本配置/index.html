Windows下对Jupyter lab进行基本配置

2020-04-11

主要是怎么配置python,以及添加tensorflow库等

  1. 下载Anaconda,安装时有2个选项（配置环境变量等等），2个都勾上。

  2. 新建一个环境，例如命名为tf1，安装jupyter lab

  3. 在新环境下，打开cmd，准备安装tensorflow库。在国内直接安装比较慢，用清华大学镜像：

pip install -i https://pypi.tuna.tsinghua.edu.cn/simpletensorflow==1.18

默认不是1.18版本，但我当时安装最新版本时，会报错。

Numpy库也可以重装下，防止报错。

  1. 在python中输入import tensorflow as tf进行测试
  1. 将这个环境配置给jupyter lab:

conda install nb_conda ipykernel

python -m ipykernel install –user –name tf1

这样jupyter lab内核选项中就多了个tf1

删除kernal: jupyter kernelspec remove kernelname

查看：jupyter kernelspec list

6.怎么方便地打开jupyter lab.可以建个bat文件，里面输入jupyter lab.将这个bat文件放在代码项目所在文件夹，这样打开后，默认就是在这个目录。

windows下配置anaconda环境时，建议不打开conda界面，在cmd中操作。界面里有点卡。其实更好的办法是忽略以上步骤，就在cmd中进行操作：

conda create –name tf1

activate tf1

conda install tensorflow 貌似会自动检测出清华以及其他镜像进行下载。。装的挺快

然后配置 jupyter:

conda install nb_conda ipykernel

python -m ipykernel install –user –name tf1
