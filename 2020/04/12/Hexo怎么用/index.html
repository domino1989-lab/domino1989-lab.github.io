Hexo怎么用

2020-04-12

Contents

  1. 怎么写新文章：
  2. 怎么更换Hexo主题

Hexo网站上有详细说明怎么安装，这里不说了。 主要是用Hexo在本地生成网页，然后把网页上传到github上。

怎么写新文章：

  1. 在站点文件夹内右键打开git bash,输入hexo new 文章名 （或者直接在站点source文件夹内新建 文章.md）
  2. 在md文件内写文章
  3. 可以hexo serve，进行测试
  4. hexo d -g 这样生成网页，并上传到github上

怎么更换Hexo主题

  1. 一般是先copy主题到themes文件夹下
  2. 更改_config.yml下theme的名字
  3. 再进行一些个性化设置

主题生成有问题时，很大原因是有些包没装，可以用npm ls –depth 0 进行查看。用npm进行安装，安装没反应时，可以改用cnpm等方法，例如：
npm install -g cnpm –registry=https://registry.npm.taobao.org
npm config set registry https://registry.npm.taobao.org
cnpm install hexo-renderer-sass –save
